● Qué usuarios existen

    Los usuarios que existen son, si es el ordenador donde hemos desarrollado el programa solo el usuario.

● Qué permisos tienen

    Cuando se descarga el programa por alguien externo a los creadores del mismo, la idea es hacer que este usuareio solo renga los permisos de ejecucion para poder empezar a usar el programa.

● Qué carpetas se usan

    Las carpetas que se usan son las diferentes carpetas de las asignaturas, en nuestro caso las carpetas: BBDD, PuchamonGris, SistemasInformaticos y Web, para poder tener las cosas ordenadas segun las especificaciones de cada asignatura.

● Dónde se guardan datos

    Los datos se guardan en nuestra base de datos (DBPuchamon) que esta en local, pero se puede ver todo el procesor de crecion de la base de datos en la carpeta BBDD para poder crearla.

● Dónde se guardan copias de seguridad

    En nuestro caso no hemos implementado copias de seguridad en nuestro proyecto.
