● Sistema operativo principal (Windows, Linux, etc.)

    El SO principal sera Windows aunque se pueda ejecutar en los otros SO ya que la app sera multiplataforma

● Versión recomendada

    La ultima version disponible que ese SO tenga, aunque el minimo que se pide es Windows 10 o superior, Linux (distribución moderna) y macOS 10.13 o superior.

● Por qué ese y no otro 

    Basicamente porque sera la que mejor desempeñe la funcion de ejecutar el juego y sera la mas rapida.
