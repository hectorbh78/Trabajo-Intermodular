● Qué hay que instalar

    Hay que instalar Java Runtime Environment (JRE) para poder ejecutar el programa ya que es una clase en java, depues bajarse el archivo y ejecutarlo.

● En qué orden

    El orden de instalacion es el anterior mencionado, primero se instala el JRE y despues se baja el programa para poder ejecutarse, para hacer esto habria que abrir la terminal del SO correspondiente(buscar cual es y como abrirla), y despues escribir la ruta hasta llegar a el programa y escribir java -jar nombreJuego.jar. .

● Qué dependencias necesita tu proyecto

    Las cosas dependientes que tiene nuestro proyecto seria que en le caso de jugarse en PC se necesita un teclado para poder jugar.

● Qué configuraciones básicas son necesarias

    La configuracion estandar el tu SO tenga es suficiente junto conla instalacion de la JRE y el programa.
