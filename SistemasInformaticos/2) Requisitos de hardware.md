● CPU mínima y recomendada

    Minimo de 2 nucleos (como Intel Core i3)

● RAM mínima y recomendada

    Minimo de 4GB de RAM 

● Almacenamiento necesario

    ALmacenamiento minimo de unso 500MB o 1GB.

● Periféricos si aplican

    El unico periferico necesario es el teclado para poder jugar al juego, aunque un raton podria ser recomendable.
