● ¿En qué tipo de equipo debería ejecutarse tu aplicación?

    Nuestra aplicacion puede descargarse en casi cualquier SO, ya que nuestro objetivo es que sea multiplataforma y que consumas los menores recursos posibles

● ¿Qué sistema operativo necesita?

    En principio no es necesario ningun SO en concreto, si la aplicaion realmente es funcional deberia funcionar en cualquier tipo de SO

● ¿Qué recursos mínimos y recomendados requiere?

    Por parte de los requisitos minimos necesarios, dependiendo del SO se necesitaria una version de: Windows 10 o superior, Linux (deberia ser una distro moderna) y en cuanto a mac, seria macOS 10.13 o superior.a

    Tambien un minimo de un procesador de 2 nucleos por parte del procesador, memoria ram de 4GB minima, una grafica integrada como Intel HD o parecida para el renderizado del juego en 2D si lo tuviese. Ademas un almacenamiento minimo de unos 500MB o 1GB para poder cargar las imagenes de los puchamones y poder guardar el progreso, una resolucion recomendada de unos 1280x720 (lo minimo en PC). 

    Como se puede ver son requisitos bastantes faciles de cumplir para poder ejectar el juego de puchamon, asi que la mayoria de gente podra disfrutar de el, puesto que nuestro enfoque es que lo juegue especialmente la gente que no puede permitirse comprar un pokemon de verdad.

● ¿Cómo se instala?

    Ya que es un proyecto en java debera de tenrse primero instalado Java Runtime Environment (JRE) para poder ejecutar el juego, despues instalar el juego bajandoselo y ejecutarlo

● ¿Cómo se ejecuta?

    Si ya tenemos el juego instalado podemos simplemente hacer doble click en el y se abrrira correctamente, tambien podemos hacerlo desde la cmd escribimos la ruta del juego y ejecutamos el sigiente comando java -jar nombreJuego.jar. 

● ¿Cómo se mantiene?

    Asegurandose de que funcione en los diferentes SO para los que esta hecha, actualizandola periodicamente con nuevo contenido para ofrecer a los jugadores, arreglar errores si los hubiera par ana mejor experiencia de juego y optimizar el juego si tuviera algun problema de bajones de FPS o algo parecido.

● ¿Cómo se protege mínimamente?

    Se encriptaria el codigo para que ningun usuario puediera modificarlo ni leerlo, asi el codigo estaria a salvo de gente deshonesta.
